# 📖 English Visual Vocabulary

DK 风格英语视觉词汇学习 PWA 应用。

**10 个场景 · 275 个核心词汇 · 离线可用 · 音频播放 · 收藏复习**

## 功能

- 🏡 居家生活（家具、房间、家务）
- 🍳 厨房烹饪（厨具、烹饪动作）
- 🛒 购物服装（服饰、购物流程）
- 🍽️ 外出就餐（点餐、餐具）
- 💼 办公室工作（办公用品、职场）
- ✈️ 旅行交通（机场、交通方式）
- 🌤️ 天气自然（天气、四季）
- 🏥 健康医疗（身体、看病）
- 🏃 运动休闲（运动、爱好）
- 📚 学习教育（科目、学习）

## 技术栈

纯静态网页（HTML + CSS + JS）+ PWA：
- 零依赖，双击即可运行
- Web Speech API 语音朗读
- PWA 支持安装到手机主屏幕，离线可用
- LocalStorage 持久化收藏数据
- 响应式设计，支持暗色模式
- GitHub Pages 免费托管

## 使用方式

- **Web**: 访问 GitHub Pages URL
- **手机**: 用浏览器打开 → 添加到主屏幕 → 像原生 App 一样使用
- **桌面**: 双击 `index.html` 即可
